<h2>Overview</h2>

<p>
SPACE is a social communication platform designed to give users a safe space
to publish posts, discover and connect with new friends, and chat with
others in real time. Built as part of NUS Orbital — the School of
Computing's self-driven summer programming course — the project spans both
PC (web) and Android platforms, with a shared backend powering content
moderation, social discovery, and messaging features across devices.
</p>

<h2>Abstract</h2>

<p>
SPACE addresses the need for a social media experience that prioritizes user
safety alongside the core features expected of a modern platform: publishing
posts, meeting new people, and chatting with friends. The platform was
developed using Angular for the frontend and Firebase for authentication,
real-time data storage, and backend services, allowing the same core logic
to support both a PC web client and an Android application. Through NUS
Orbital, the project was evaluated against a cohort of roughly five hundred
students and was awarded the Advanced Certificate, placing in the top 5% of
all participating teams.
</p>
